## Quote

### Raw notation

```
> foo
> bar
> baz
```

> foo
> bar
> baz

### Block notation

It is recommended that "quote" block like "table".

@@@

```quote
> **foo**
>
> //bar//
> > __baz__
> > **baz**
> > baz3
> > > gab
> > >
> nano
> desu
> > > > foo
> > bar
```
