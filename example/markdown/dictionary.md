## Dictionary "="

### Store string in the dictionary

```
[key]: value        // notation 1

=[key2](value2)     // notation another
```

[key]: value

=[key2](value2)

### Simple load "="

```
=[key]

=[key2]
```

=[key]

=[key2]

### Ruby load "+"

```
+[key]

+[key2]
```

+[key]

+[key2]

### Reverse ruby load "-"

```
-[key]

-[key2]
```
